learning :
	step0:
		you should delete the file named "dist"
		
	step1:
		after done the step0 ,if you open the file named "index.html" ,you can see nothing in the page but white panel.
		you should install the dependences by doing "npm install" on your console.
	step2:
		after done the step1 ,you can see the same effect on the page as the step0.
		but ,if you want to delete the "node_modules",and see the same effect,you should do one more step by running 
		"npm run build " on your console.this is the result you see in the step0.
